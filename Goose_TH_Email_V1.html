To use:
This template renders Total Hockey Marketing emails to be sent through Bronto or Smartfocus.
Type in the "alt" text (typically the subject line): this will appear in the users window if they have images turned off when viewing.

When inserting links, make sure to use the full HTTP url, followed by Google parameter tags. When inserting the link for the image, make sure the image is 640x900 and is available in the /ui/ directory. The tool will show a broken image if the asset is not in the /ui/. Click "GET CODE."

You can view your inputs into the HTML generated code. Once code is copied, place this into Bronto or SmartFocus and test away!
